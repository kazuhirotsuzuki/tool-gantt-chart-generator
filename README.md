# Overview
This project contains a Python Jupyter Notebook that generates a project schedule table and a Gantt chart from an Excel spreadsheet. It processes timeline data and outputs a single HTML file showing project tasks, start times, and assigned teams.

Key features include:
* **Data Processing**: Standardises time formats, handles missing start times, and calculates plotting dates based on the project date.
* **Gantt Chart**: Creates an interactive Gantt chart using `plotly`, colour-coded by assignee.
* **Schedule Table**: Generates a schedule table grouped by task category using `great_tables`.
* **HTML Output**: Combines the table and chart into one HTML file.

# File Structure
The project uses the following files:

* **`GanttChartGenerator.ipynb`**: The main Jupyter Notebook. It uses `pandas` for data manipulation, `great_tables` for the table, and `plotly.express` for the chart.
* **`Timeline.xlsx` (Input Data)**: The input Excel file. The script automatically reads the first `.xlsx` file it finds in the directory.
* **`Timeline.html` (Output)**: The HTML document generated by the notebook containing the table and chart.

# Input Data Structure
The script reads data from two specific sheets in the input Excel file:

**1. `Information` Sheet**
Contains project variables used in the script:
* **`projectName`**: Used for HTML titles and headings.
* **`goLiveDate`**: The project date (e.g., `"2026-04-29"`), used to plot the timeline.
* **`prevTaskComment`**: Text for tasks without a strict start time (e.g., `"Before 9 a.m."`).
* **`prevTaskStart`**: A fallback start time used for plotting (e.g., `"08:00:00"`).

**2. `Timeline` Sheet**
Contains the task list. The script requires the following columns:
* `No`, `Category`, `Predecessors`, `Start`, `Finish`, `Duration`, `Task`, `CheckPoint`, `Assignee`
